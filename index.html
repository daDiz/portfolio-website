Projects

Predict the link between a group and an individual using Conditional Variational Auto-encoder (Python, Tensorflow).

Dynamic Topic Modeling

Analyze the Twitter narratives around COVID-19 response by applying a network Hawkes binomial topic model to tweets by U.S. government officials.

Network Event Prediction

Model network event sequences using hierarchical multiscale LSTM and second-order statistic loss (Python, Tensorflow).

Interpretable Hawkes Process

Develop an interpretable Hawkes process model of crime (Hawkes process with a total-variation regularization) that allows forecasts to capture near-repeat effects and spatial heterogeneity while being consumable in the form of easy-to-read score cards.

Apply YOLO to detect corn-stalk connection points from videos and predict corn heights from point patterns using traditional machine learning methods like ridge/lasso regression, KNN, XGBoost, random forests.
